# Programmatic Growth Beta Test Checklist

Use this checklist for internal validation of the controlled beta flow.

1. Choose an opportunity.
2. Detect programmatic potential.
3. Build a cluster preview.
4. Review cluster items.
5. Create blueprints.
6. Approve blueprints.
7. Create briefs.
8. Prepare draft requests.
9. Create drafts.
10. Run quality checks.
11. Convert content.
12. Run publication readiness.
13. Create a publication plan.
14. Prepare scheduled publication records.
15. Confirm that nothing has been live published.

Safety expectation: Programmatic Growth may create local assets, quality checks, content records and scheduled publication records. It must not call external destinations or publish live content automatically during beta testing.
